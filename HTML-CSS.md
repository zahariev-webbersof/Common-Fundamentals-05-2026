# Как да учите ефективно

Когато попаднете на непознат HTML елемент или CSS свойство:

1. Потърсете го в MDN.
2. Разгледайте примерите.
3. Използвайте го в собствен проект.
4. Експериментирайте с различни стойности.

Само четенето не е достатъчно. Истинското учене започва, когато започнете да създавате собствени проекти!
