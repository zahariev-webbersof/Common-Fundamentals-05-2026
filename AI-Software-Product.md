# 🔄 Understanding How Software Works

Every software application consists of multiple parts working together.

```text
User
  ↓
Front-End
  ↓
Back-End
  ↓
Database
```

### 👤 User

The person using the application.

### 🖥️ Front-End

Everything visible to the user.

Built with technologies such as:

* HTML
* CSS
* JavaScript

### ⚙️ Back-End

The server-side logic.

Responsible for:

* Processing requests
* Business logic
* Authentication
* Communication with databases

### 🗄️ Database

Stores information such as:

* Users
* Products
* Orders
* Messages
* Reservations

# 🧩 How HTML and CSS Work Together

Think of a house:

🏠 HTML = Structure

* Walls
* Doors
* Windows
* Rooms

🎨 CSS = Design

* Colors
* Furniture
* Decorations
* Layout

Without HTML there is no structure.

Without CSS the structure exists, but it looks plain.

Together they create a complete user interface.

---

# 🚀 What Happens When You Open index.html?

When you open your project:

1. The browser reads the HTML file.
2. The browser loads the CSS file.
3. HTML creates the page structure.
4. CSS styles the page.
5. The browser displays the final result.

```text
index.html
     +
style.css
     ↓
Browser
     ↓
Web Page
```

This is the simplest example of multiple technologies working together to create a software product.
