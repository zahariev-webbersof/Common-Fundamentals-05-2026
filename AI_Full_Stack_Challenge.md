# 🚀 AI Full-Stack Mini Challenge

> **Programming Fundamentals | AI-Assisted Development | Mini Project**

## 📚 Project Overview

Welcome to your first **AI Full-Stack Development Challenge!** 🎉

In today's software industry, developers don't work alone—they work **with AI**.

The purpose of this project is **not** to write hundreds of lines of code from memory.

Instead, your goal is to:

* 🤖 Learn how to use AI as a software development assistant
* 🧠 Understand the generated code
* 🌐 Connect a Front-End with a Back-End
* 💡 Experience how modern web applications are built

This project is language-independent, so you can use:

* 🐍 Python
* ☕ Java
* 💙 C#
* 💛 JavaScript (Node.js)
* ...or any backend language approved by your instructor.

---

# 🎯 Your Mission

Create a modern website about **one topic of your choice**.

Choose one of the following ideas (or create your own):

* 🚗 Favorite Car
* 🎬 Favorite Movie
* 📚 Favorite Book
* 🎮 Favorite Video Game
* 🏀 Favorite Sport
* 🐶 Favorite Animal
* 🌍 Favorite Country
* 🏖️ Dream Vacation
* 🍕 Favorite Food
* 🎵 Favorite Music Artist

---

# 🛠️ Step 1 – Create the Front-End

Ask your AI assistant:

> Create a modern responsive website using HTML, CSS and Bootstrap about **[your topic]**.
>
> Include:
>
> * Navigation Bar
> * Hero Section
> * Image
> * Description
> * Interesting Facts
> * Footer
>
> Use a clean and modern design.

💡 Save the generated files.

---

# 🌐 Step 2 – Create the Back-End

Ask AI to generate a simple backend using **your preferred programming language**.

Examples:

### Python

> Create a Flask application that serves my website.

### Java

> Create a Spring Boot application that serves my website.

### C#

> Create an ASP.NET Core application that serves my website.

### JavaScript

> Create an Express.js application that serves my website.

---

# 🔄 Step 3 – Connect the Back-End with the Front-End

Instead of hardcoding all the information inside the HTML page, ask AI:

> Move the title, description and interesting facts into backend variables and display them dynamically in the HTML page.

🎉 Congratulations!

You have just connected your **Front-End** and **Back-End**.

---

# 📄 Step 4 – Add Another Page

Ask AI:

> Create an About page and connect it to the navigation menu.

Your application should now contain at least **two pages**.

---

# 📬 Step 5 – Contact Form

Ask AI:

> Create a Contact page with:
>
> * Name
> * Email
> * Message
>
> When the user submits the form, display a "Thank You" message.

You are encouraged to use AI throughout the project.

You may use:

* ✅ ChatGPT
* ✅ GitHub Copilot
* ✅ Gemini
* ✅ Claude
* ✅ Any other AI coding assistant

* Ask AI to explain it.
* Debug it.
* Improve it.
* Learn from it.
